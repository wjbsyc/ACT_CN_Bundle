# ACT_CN_Bundle

ACT(原版)+国服插件整合，自用

----

**这是一个基于原版[ACT](https://github.com/EQAditu/AdvancedCombatTracker)的国服插件整合包，自用，也适用于出于各种原因需要用原版ACT的国服玩家。**



----

##### 该整合包整合了以下项目，均为原版二进制文件或基于原仓库代码编译的二进制文件，未经过任何修改。

* [Advanced Combat Tracker](https://github.com/EQAditu/AdvancedCombatTracker)

* [NewMoe-Technology/FFXIV_ACT_Plugin_CN](https://github.com/NewMoe-Technology/FFXIV_ACT_Plugin_CN)

* [NewMoe-Technology/OverlayPlugin](https://github.com/NewMoe-Technology/OverlayPlugin)

* [NewMoe-Technology/cactbot](https://github.com/NewMoe-Technology/cactbot)

* [ACT.FoxTTS](https://github.com/Noisyfox/ACT.FoxTTS)

* [PostNamazu](https://github.com/Natsukage/PostNamazu)

* [StarlightBreaker](https://github.com/Loskh/StarlightBreaker)

* [enilia-overlay-kai](https://github.com/wjbsyc/enilia-overlay-kai)

* [UWU_auto_support](https://github.com/wjbsyc/UWU_auto_support)



----

### 使用方式：

请至[Releases](https://github.com/wjbsyc/ACT_CN_Bundle/releases) 页面下载，安装好ACT后，手动加载整合包内插件
